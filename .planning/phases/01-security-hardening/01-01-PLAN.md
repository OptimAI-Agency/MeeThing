**1. `supabase/functions/_shared/crypto.ts`** (per D-01, D-02, D-03):

Create a module exporting `encrypt(plaintext: string): Promise<string>` and `decrypt(ciphertext: string): Promise<string>`.

Implementation details:
- Algorithm: `AES-GCM` with 12-byte (96-bit) random IV per NIST recommendation
- Key source: `Deno.env.get("ENCRYPTION_KEY")` — must be a 64-character hex string (256 bits)
- `getKey()` helper: reads hex from env, validates length is exactly 64, converts hex to `Uint8Array`, imports via `crypto.subtle.importKey("raw", ..., { name: "AES-GCM" }, false, ["encrypt", "decrypt"])`
- `encrypt()`: generates random IV via `crypto.getRandomValues(new Uint8Array(12))`, encodes plaintext with `TextEncoder`, encrypts with `crypto.subtle.encrypt({ name: "AES-GCM", iv }, key, encoded)`, prepends IV to ciphertext, returns base64 string via `btoa(String.fromCharCode(...))`
- `decrypt()`: decodes base64 to bytes, splits first 12 bytes as IV and remainder as ciphertext+tag, decrypts with `crypto.subtle.decrypt({ name: "AES-GCM", iv }, key, data)`, returns decoded string via `TextDecoder`
- Helper functions: `hexToBytes(hex: string): Uint8Array`, `bytesToBase64(bytes: Uint8Array): string`, `base64ToBytes(b64: string): Uint8Array`
- If `ENCRYPTION_KEY` is not set, throw `new Error("ENCRYPTION_KEY not set")`
- If `ENCRYPTION_KEY` length is not 64, throw `new Error("ENCRYPTION_KEY must be 64 hex chars (256 bits)")`

Use the exact implementation from 01-RESEARCH.md Pattern 1.

**2. `supabase/functions/_shared/cors.ts`** (per D-09, D-10, D-11):

Create a module exporting `getCorsHeaders(): Record<string, string>` and `handleCorsPreflightIfNeeded(req: Request): Response | null`.

Implementation details:
- `getAllowedOrigin()` internal helper: reads `Deno.env.get("ALLOWED_ORIGIN")`, throws `new Error("ALLOWED_ORIGIN environment variable is not set — refusing to serve with wildcard CORS")` if not set
- `getCorsHeaders()`: returns `{ "Access-Control-Allow-Origin": getAllowedOrigin(), "Access-Control-Allow-Headers": "authorization, x-client-info, apikey, content-type" }`
- `handleCorsPreflightIfNeeded(req)`: if `req.method === "OPTIONS"`, returns `new Response("ok", { headers: getCorsHeaders() })`, otherwise returns `null`

Use the exact implementation from 01-RESEARCH.md Pattern 2.
